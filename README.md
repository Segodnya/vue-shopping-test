# vue-shopping-test

Средствами Vue.js реализовано небольшое SPA приложение для ведения списка продуктов:

- приложение состоит из 1ой страницы;
- на странице отображается список всех продуктов (дополнительно реализована возможность очищать список);
- имеется возможность добавить в список новый продукт;
- для каждого продукта имеется возможность “вычеркнуть” его, после чего название продукта становится зачеркнутым и продукт помещается в конец списка (также есть возможность "отменить" вычеркивание продукта, при этом вычеркнутые продукты все еще останутся в конце списка);
- все действия на сайте происходят без перезагрузки страницы;
- после перезагрузки страницы состояние списка продуктов сохраняется.
